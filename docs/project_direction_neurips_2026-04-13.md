# Project direction note (2026-04-13)

## Core guidance

The project should not drift into broad or weak searches just because they are adjacent to adaptive reasoning or LLM test-time compute.

The goal is to produce a meaningful and defensible contribution for a NeurIPS-style paper, not merely to collect loosely related ideas.

## Practical rule

Before pursuing a step, ask:

> Does this improve the chance of a clear, nontrivial, publishable contribution at the intersection of theoretical computer science and AI?

If the answer is unclear, the step should be deprioritized.

## Implications

1. Prefer depth over breadth.
2. Prefer contribution-oriented searches.
3. Avoid unfocused adjacent work.
4. Preserve manuscript relevance in notes, coding, and experiments.

## Current best contribution path

The strongest current direction is:
- fixed-budget adaptive compute allocation for LLM reasoning
- local learning target based on continuation value / value of computation
- theory target connecting local scoring error to global allocation regret or performance loss
- manuscript framing grounded in metareasoning and fixed-budget allocation

## Working priority order

Prefer next steps that improve:
1. theorem quality
2. target alignment
3. novelty clarity
4. manuscript framing
5. evidence for a NeurIPS-level contribution
