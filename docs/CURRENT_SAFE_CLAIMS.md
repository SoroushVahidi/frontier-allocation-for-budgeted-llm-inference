# Current safe claims (canonical)

## Safe to claim now

- The repository is a strong research platform for fixed-budget adaptive test-time compute allocation.
- Cross-controller frontier allocation is a clear and distinct framing.
- Pairwise BT branch scoring is a strong active learned direction.
- External warm-start and reliability-aware variants are promising but mixed.
- The current bottleneck is supervision target quality / proxy-label mismatch.
- The most promising near-term controller direction is budget-conditioned binary stop-vs-act with uncertainty-aware handling.

## Not safe to claim yet

- That any current learned controller is a robust universal winner.
- That external warm-start or reliability-weighted variants are settled canonical winners.
- That proxy labels already approximate true oracle marginal utility sufficiently well.
- That heavier models alone will resolve current weaknesses.
- That real-model evidence is already broad and decisive.

## Preferred wording

Prefer language such as:
- “suggests”, “is consistent with”, “promising but mixed”,
- “proxy-label mismatch”, “approximate marginal labels”,
- “active-development method maturity”,
- “bounded matched comparisons”.

## Wording to avoid

Avoid language such as:
- “solves”, “final”, “robust winner”, “universally better”,
- “oracle marginal labels” for heuristic/proxy labels,
- “only scale is missing”.
