# Claim-Safety Statistical Table Note

This note accompanies `outputs/paper_tables/table_claim_safety_statistical_tests.csv` and
`outputs/paper_tables/table_claim_safety_statistical_tests.tex`.

## Safe manuscript wording

- `strict_f3` is **not statistically established** as decisively better than `strict_gate1_cap_k6` across the paper-facing matched-surface slices.
- The safe claim is that the leading frontier-allocation variants form a **close top internal cluster**, and the exact winner is **surface-dependent**.
- Comparisons against near-direct external baselines can be stronger on some matched-surface dataset/budget slices, but these should still be stated with action-budget and matched-substrate caveats.

## Forbidden wording

- “Strict-F3 statistically dominates Gate1.”
- “Frontier allocation universally dominates external baselines.”
