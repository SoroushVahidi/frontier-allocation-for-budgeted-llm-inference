# Direct L1 Anchor Patch Report (2026-05-10)

## Problem Targeted
**Frontier Collapse / Gold Absent**: Our analysis of 172 gold-absent cases revealed that 96% of these failures show "frontier collapse," where the frontier search fails to explore semantically diverse reasoning paths and instead converges on a single (often wrong) answer group.

## Literature Motivation
Recent literature on LLM reasoning (e.g., DiVeRSe, diverse anchored generation) suggests that when the correct answer is absent from the sampled pool, selection alone is insufficient. The recommended direction is to ensure **generation diversity before selection** by using multiple reasoning anchors or direct CoT seeds, then applying filtering/selection only after the pool is broad enough.

- **Reduced Discovery Failure**: Ensures that at least one "direct" reasoning path (often more reliable than shallow frontier expansions) is always available for selection.
