\caption{Overview of the Failure-Trace Allocator (FTA) selective-deferral policy. The policy first checks the weak-frontier trigger; if it fires, the external majority answer is selected. Otherwise, the policy checks the direct-agreement trigger; if all external baselines unanimously disagree with the frontier answer, the unanimous external answer is selected. If neither trigger fires, the frontier answer is retained. Journal-facing trigger names map to implementation identifiers in Table~A6.}
