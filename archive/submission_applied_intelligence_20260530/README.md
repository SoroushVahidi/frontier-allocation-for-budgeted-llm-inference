# Archived Applied Intelligence submission packages (pre-20260530)

These artifacts are **not** the current Editorial Manager upload package. They are kept for provenance only.

## Why they were archived

Applied Intelligence required a **single self-contained LaTeX source file**. Older packages in this folder contained **many separate `.tex` files** (flat layout or multi-file tree). Uploading them caused Editorial Manager to compile extra fragments and produce an incorrect merged PDF.

## Current upload package (use instead)

**Root of repository:**

`applied_intelligence_fta_single_tex_source_20260530.zip`

That zip contains exactly one `.tex` file (`submission_applied_intelligence_single_tex/main.tex`), `refs.bib`, `svjour3.cls`, and four figure PNGs—**no PDF**.
