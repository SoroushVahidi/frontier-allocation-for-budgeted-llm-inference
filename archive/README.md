# Archive

This directory is reserved for preserved historical material, superseded notes, and provenance-only assets that should remain available without competing with the current canonical project path.

## Intended use

Place here:
- superseded paper-track material,
- older direction-specific notes,
- provenance-only experiment summaries,
- historical support files that should not be read as current project guidance.

## Interpretation rule

Anything in `archive/` should be treated as:
- useful for provenance,
- useful for historical comparison,
- not the default interpretation of the repository.

For the current project path, start instead with:
- `README.md`
- `docs/CANONICAL_START_HERE.md`
- `docs/REPO_MAP.md`
