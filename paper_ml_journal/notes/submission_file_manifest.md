# Submission File Manifest (Springer MLJ Portal)

## Main manuscript

- Upload a LaTeX ZIP containing editable manuscript source.
- Include compiled PDF if the portal requests it explicitly or generates it after source upload.

## Figures and tables (separate files)

- Optional at initial submission unless explicitly required by the portal.
- If requested, provide separate high-resolution figure assets:
  - Figure 2 PDF/PNG
  - Figure 3 PDF/PNG
  - Figure 1 source and/or rendered PDF
- Table `.tex` sources may be included in the manuscript ZIP; separate table files are optional unless requested.

## Supplementary material (ESM)

- Optional and only for publishable supplementary content.
- If submitted, provide a curated reproducibility/artifact ZIP after manual review.

## Related files (editor/reviewer assessment)

- Upload related conference manuscript PDF if available.
- Upload relationship/overlap disclosure note.
- Treat these as review-assessment files, not publication content.

## Cover letter

- Upload cover letter text derived from `notes/mlj_cover_letter_draft.md`.
- Submit as plain text or `.txt`/`.md` according to portal acceptance.
