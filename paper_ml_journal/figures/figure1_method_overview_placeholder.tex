\caption{Failure-trace-guided allocator (FIX-2+FIX-4). The allocator first checks
the weak-frontier trigger (Low-Depth Guard, FIX-2): if it fires, the policy uses the
external majority answer. Otherwise it checks the direct-agreement trigger
(External-Consensus Fallback, FIX-4): when all external baselines unanimously disagree
with the frontier answer, the policy uses the external unanimous answer. If neither
trigger fires, the frontier answer is retained. No gold labels or correctness signals
are accessed at any step. Paper-facing trigger names map to implementation identifiers
in Table~\ref{tab:name_mapping}.}
