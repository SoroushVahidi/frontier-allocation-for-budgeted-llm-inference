# Historical scripts index

This page groups script entry points that are preserved for provenance but are no longer part of the current default project path.

## Archived historical scripts

These scripts have been moved under `archive/historical_scripts/`:

- `archive/historical_scripts/run_heavy_real_routing_eval.sh`
- `archive/historical_scripts/run_final_manuscript_eval.sh`

## How to interpret them

These scripts are useful for:
- provenance,
- understanding older evaluation workflows,
- and reconstructing parts of the superseded binary-routing / older-manuscript lines.

These scripts are **not** the default starting point for the current repository.

## Current default path instead

Use these pages first:
- `CANONICAL_START_HERE.md`
- `README.md`
- `../docs/CANONICAL_START_HERE.md`

## Practical rule

If a script primarily supports:
- the old binary revise-routing track,
- older manuscript-facing controller evaluation,
- or superseded real-routing matrices,
then it belongs in the historical/provenance path rather than the canonical scripts path.
